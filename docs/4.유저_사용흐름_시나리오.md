# 유저 사용 흐름 시나리오

유저 사용 흐름 시나리오

시나리오 목적
이 시나리오는 사용자가 앱에 처음 가입한 뒤, MVP에 포함된 핵심 기능을 모두 한 번씩 경험하는 흐름을 설명한다.

시나리오  1. 최초 가입부터 첫 오늘의 미션 완료까지

1단계. 앱 실행
사용자가 앱을 처음 접속(실행)한다.
앱은 로그인/가입 화면을 보여준다.

2단계. 이메일 가입
사용자는 이메일을 입력하고 가입 진행한다.
가입이 완료되면 최초 접속 사용자이므로 홈으로 바로 가지 않고, 프로필/관심사 설정 화면으로 이동한다.
프로필/관심사 설정화면에서 프로필사진을 제외한 나머지 필드를 모두 설정(입력)하지않으면 시작하기 버튼은 비활성 상태로두고 , 설정하면 시작하기 버튼이 활성화되어 홈화면으로 이동할수있다.

3단계. 프로필/관심사 설정
사용자는 아래 정보를 입력한다.
프로필 사진 선택 또는 기본 이미지 유지 (선택)
닉네임 입력 (필수입력)
나이 선택(필수입력)
국가 선택(필수입력)
한 줄 프로필 입력(필수입력)
관심사 선택 (필수입력)
필수 항목(프로필사진섹션만선택입력)을 모두 입력하면 시작하기 버튼이 활성화된다.
사용자가 시작하기를 누르면 프로필이 저장되고 홈으로 이동한다.

시나리오 2. 홈에서 매칭 시작하기

4단계. 홈 진입
사용자는 홈 화면에 도착한다.
홈에는 아래 두 핵심 영역이 보인다.
오늘의 미션 섹션
Board 섹션
오늘의 미션 영역에는
오늘의 미션 제목
매칭 상태
매칭 시작 버튼 이 표시된다.

5단계. 매칭 시작
사용자가 매칭 시작 버튼을 누른다.
그러면 매칭 시작 확인 모달이 등장한다.
매칭 시작 확인 모달(최종 카피):
- 제목: 오늘의 팀을 찾을까요?
- 본문: 앱을 닫아도 매칭되면 알려드려요.
- 버튼: 취소 / 시작하기
사용자가 시작하기를 누르면 매칭 요청이 생성된다.

6단계. 매칭 대기
홈의 오늘의 미션 영역 상태가 매칭 중...으로 바뀐다.
버튼은 비활성화된다.
사용자는 앱을 닫아도 상관없이 매칭은 진행된다

7단계. 매칭 완료 푸시 수신
매칭이 완료되면 푸시 알림이 온다.
예:
Japan 팀원과 매칭 완료! 오늘의 미션에 참여해보세요
사용자가 푸시를 누르거나 다시 앱을 열면 홈으로 들어온다.
홈의 오늘의 미션 영역 버튼은(매칭중......에서 오늘의 미션 참여로 바뀌고 활성화되어있어야한다.)

시나리오 3. 오늘의 팀 미션 확인 및 제출

8단계. 오늘의 팀 미션 화면 진입
사용자가 홈의 오늘의 미션 참여 버튼을 누른다.
오늘의 팀 미션 화면으로 이동한다.
이 화면에는 아래가 보인다.
오늘의 미션 제목/설명
주의사항 i 아이콘
내 팀 정보
진행 상태
상태에 따른 버튼
현재 사용자는 아직 제출하지 않았기 때문에 버튼은 미션하기다.

9단계. 미션 참여 화면 진입
사용자가 미션하기 버튼을 누른다.
미션 참여 화면으로 이동한다.

10단계. 사진 업로드 및 한 줄 작성
사용자는
사진영역을 터치하여 갤러리에서 사진을 선택하고 1장 업로드 하고
한 줄 설명 입력 을 완료한다.
예:
사진: 카페 사진
설명: 퇴근 전에 카페에서 잠깐 정리 중이에요
모든 입력이 완료되면 제출하기 버튼이 활성화된다.

11단계. 제출 완료
사용자가 제출하기를 누른다.
제출 성공 시 토스트가 노출된다.
예:
제출이 완료되었어요
이후 사용자는 오늘의 팀 미션 화면으로 돌아온다.

12단계. 상대 대기 상태 확인
오늘의 팀 미션 화면에서 상태가 바뀐다.
나: 업로드 완료
상대: 아직 업로드 전
진행률: 1/2
버튼: 결과 보기 비활성
사용자는 상대 팀원의 제출을 기다린다.

시나리오 4. 결과 보기 및 반응 남기기

13단계. 상대 제출 완료 알림 수신
상대가 제출하면 푸시 알림이 온다.
예:
상대 팀원이 답변을 완료했어요
이제 팀 결과를 볼 수 있어요

14단계. 결과 보기 화면 진입
사용자가 알림을 누르거나 오늘의 팀 미션 화면으로 다시 들어간다.
이제 결과 보기 버튼이 활성화되어 있다.
사용자가 결과 보기를 누르면 결과 보기 화면으로 이동한다.

15단계. 결과 카드 확인
사용자는 내 카드와 상대 카드가 함께 묶인 결과 카드를 본다.
국가/닉네임/프로필
사진
한 줄 설명
한 줄 비교 문구

16단계. 공통 반응 남기기
사용자는 공통 아이콘 반응 중 하나를 누른다.
예:
👀 흥미로워요
반응 수가 즉시 반영된다.

17단계. 표현 반응 남기기
사용자는 표현 선택 버튼을 누른다.
표현 선택 바텀시트가 열린다.
예:
분위기 좋아 보여요
우리랑 비슷하네요
가보고 싶어요
하나를 선택하고 완료한다.
토스트가 표시된다.
예:
표현을 남겼어요

시나리오 5. Board 탐색 및 저장 기능 사용

18단계. 홈으로 돌아가기
사용자는 홈으로 돌아온다.

19단계. Board 진입
홈의 Board 미리보기 또는 하단 탭 Board를 누른다.
Board 화면으로 이동한다.
Board에는 다른 팀의 결과 카드들이 목록/그리드 형태로 보인다.

20단계. 다른 팀 카드 보기
사용자는 흥미로운 카드 하나를 누른다.
동일한 결과 보기 화면으로 이동한다.
이 경우는 내 팀 결과가 아니라 다른 팀 결과다.

21단계. 카드 저장
사용자는 저장 기능이 있으면 해당 카드를 저장한다.
토스트가 표시된다.
예:
카드가 저장되었어요 (마이페이지 내활동섹션 저장한카드에서볼수있다)

22단계. 다른 팀 카드에 반응
사용자는 그 카드에도 공통 반응 또는 표현 반응을 남긴다.

시나리오 6. World 탭 사용

23단계. World 진입
사용자는 하단 탭의 World를 누른다.
World 화면으로 이동한다.
이 화면에서 사용자는 아래 정보를 본다.
오늘 참여 국가 수
생성된 팀 수
올라온 카드 수
활발한 국가
지금 주목받는 카드

24단계. 주목받는 카드 보기
사용자는 World 화면의 주목받는 카드 중 하나를 누른다.
(타팀)결과 보기 화면으로 이동한다.

시나리오 7. My 탭 사용

25단계. My 진입
사용자는 하단 탭의 My를 누른다.
마이페이지로 이동한다.
이 화면에서 사용자는 아래를 본다.
내 프로필
활동 요약
내 결과 카드
저장한 카드
설정 메뉴

26단계. 내 결과 카드 확인
사용자는 내 결과 카드를 누른다.
내가 참여했던 결과 카드 목록 화면으로 이동한다.
사용자는 방금 참여한 오늘의 결과 카드를 다시 확인할 수 있다.

27단계. 저장한 카드 확인
사용자는 저장한 카드를 누른다.
Board에서 저장했던 카드 목록을 본다.

시나리오 8. 프로필 수정 및 설정 사용

28단계. 프로필 편집
사용자는 마이페이지 상단의 편집 아이콘을 누른다.
프로필 편집 화면으로 이동한다.
사용자는
닉네임
한 줄 프로필
관심사 등을 수정하고 저장하기를 누른다.
토스트 예:
프로필이 저장되었어요

29단계. 알림 설정
사용자는 마이페이지에서 알림 설정을 누른다.
알림 설정 화면으로 이동한다.
사용자는
매칭 완료 알림
상대 제출 완료 알림
새 반응 알림 등을 on/off 할 수 있다.

30단계. 언어 설정
사용자는 언어 설정 화면으로 이동해 앱 언어 또는 번역 언어를 조정할 수 있다.

31단계. 문의하기
사용자는 문의하기 화면으로 이동해 오류 제보나 문의를 남길 수 있다.

시나리오 9. 알림 페이지 사용

32단계. 알림 페이지 진입
사용자는 상단 알림 아이콘 또는 알림 페이지를 통해 인앱 알림 기록을 확인한다.
예:
Japan 팀원과 매칭되었어요
상대 팀원이 답변을 완료했어요
내 카드에 새로운 반응이 도착했어요

33단계. 알림 클릭
사용자는 알림 항목을 눌러 해당 화면으로 이동한다.
매칭 완료 → 오늘의 팀 미션
상대 제출 완료 → 결과 보기
새 반응 도착 → 결과 보기
최종 요약 흐름
가입
→ 프로필/관심사 설정
→ 홈
→ 매칭 시작
→ 매칭 완료
→ 오늘의 팀 미션
→ 미션 참여
→ 제출 완료
→ 결과 보기
→ Board 탐색
→ 다른 카드 저장/반응
→ World 확인
→ My에서 내 결과 카드 / 저장한 카드 확인
→ 프로필 편집 / 설정 / 문의 / 알림 확인
